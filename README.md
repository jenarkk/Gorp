## 📜 Features
- Leap Announcer
- Anti Monkey
- Party Commands
- Auto GFS

## 🌐 How to install
1. Download [`ChatTriggers`] and place it into your mods folder.
1. Download the module from the [`Releases`] tab.
2. Move it to your ChatTriggers modules folder.
3. Run the `/ct load` command.
